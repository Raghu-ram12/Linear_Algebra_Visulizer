Here are some ideas ordered from easier to harder:
Easy

Display vector components and magnitude as a label next to each vector
Shear transformation (you already have the math in transformations.py)
Limit rotation input to 0-360 with validation

Medium

Draw the resultant vector when you add two selected vectors
Show the angle between two selected vectors
Dot product display between two selected vectors
Animate scaling like you animate rotation
Show unit vector of selected vector

Hard

Matrix transformation — let user input a custom 2x2 matrix and apply it to a vector
Show the span of a vector (draw the full line through origin)
Linear independence checker — highlight when two vectors are linearly dependent (cross product = 0)
Animate the transformation from identity matrix to a custom matrix, showing how the basis vectors transform (this is the 3Blue1Brown style visualization)

Challenging

Extend to 3D vectors using projection onto a 2D canvas
Eigenvalue/eigenvector visualization — show eigenvectors staying on their span after transformation
Draw the transformation of the entire grid when a matrix is applied, not just a single vector

The 3Blue1Brown style grid transformation would be the most visually impressive and directly uses all the matrix math you've already built
